Írd meg az alábbi három függvényt, majd a főprogramban hívd meg mindegyiket ugyanarra a beolvasott listára!

  1. osszeg(lista) — visszaadja az elemek összegét
  2. megszamlal(lista, ertek) — visszaadja, hányszor szerepel az adott érték

Fogalmak: def, paraméterátadás, return, függvény meghívása. Minden függvény egy feladatot csinál, és return-nal ad vissza eredményt.
Python 
def osszeg(lista):
    s = 0
    for x in lista:
        s += x
    return s

def megszamlal(lista, ertek):
    db = 0
    for x in lista:
        if x == ertek:
            db += 1
    return db

def legnagyobb(lista):
    maxe = lista[0]
    for x in lista:
        if x > maxe:
            maxe = x
    return maxe

MintaÁtlag: 3.50

Átlag felettiek: Nagy Béla, Kiss Réka
Fogalmak: Függvény hív függvényt (atlag_felett meghívhatja az atlag-ot). Párhuzamos listák indexelése.
Python 
def atlag(jegyek):
    ossz = 0
    for j in jegyek:
        ossz += j
    return ossz / len(jegyek)

def legjobb_tanulo(nevek, jegyek):
    legjobb_i = 0
    for i in range(len(jegyek)):
        if jegyek[i] > jegyek[legjobb_i]:
            legjobb_i = i
    return nevek[legjobb_i]

def atlag_felett(nevek, jegyek):
    a = atlag(jegyek)
    eredmeny = []
    for i in range(len(jegyek)):
        if jegyek[i] > a:
            eredmeny.append(nevek[i])
    return eredmeny

Termékek nyilvántartása
összetett feladattöbb algoritmusfüggvények
▼

Kérj be n termék nevét és árát (egész Ft)! Függvényekkel oldd meg:

  1. legdragabb(nevek, arak) — a legdrágább termék neve és ára
  2. olcso_termekek(nevek, arak, hatar) — azok neve, amelyek olcsóbbak a határárnál
  3. atlagos_ar(arak) — az átlagos ár egész számra kerekítve
  4. dragabb_az_atlagnal(nevek, arak) — hány termék drágább az átlagnál?
Fogalmak: Összes előző fogalom kombinálva. A 4. függvény a 3.-at hívja meg belül. Érettségi szintű összetettség.
Python 
def legdragabb(nevek, arak):
    maxi = 0
    for i in range(len(arak)):
        if arak[i] > arak[maxi]:
            maxi = i
    return nevek[maxi], arak[maxi]

def olcso_termekek(nevek, arak, hatar):
    lista = []
    for i in range(len(arak)):
        if arak[i] < hatar:
            lista.append(nevek[i])
    return lista

def atlagos_ar(arak):
    ossz = 0
    for a in arak:
        ossz += a
    return ossz // len(arak)

def dragabb_az_atlagnal(nevek, arak):
    a = atlagos_ar(arak)
    db = 0
    for ar in arak:
        if ar > a:
            db += 1
    return db

# Főprogram
n = int(input("Termékek száma: "))
nevek = []
arak = []
for _ in range(n):
    nevek.append(input("Termék neve: "))
    arak.append(int(input("Ár (Ft): ")))

nev, ar = legdragabb(nevek, arak)
print(f"Legdrágább: {nev} ({ar} Ft)")

hatar = int(input("Határár (Ft): "))
print("Olcsó termékek:", olcso_termekek(nevek, arak, hatar))

print("Átlagos ár:", atlagos_ar(arak), "Ft")
print("Átlagnál drágább termékek:", dragabb_az_atlagnal(nevek, arak))
